# Smart City Data Analysis (EDA)

This project performs **Exploratory Data Analysis (EDA)** on a Smart City dataset to analyze urban infrastructure, environmental conditions, and traffic patterns across multiple cities.

## Dataset

- **1500 records × 14 features**
- Source: U.S. Government Open Data Platform
- Includes metrics such as:
  - Traffic Congestion
  - Vehicle Speed
  - PM2.5 Levels
  - Renewable Energy Usage
  - Noise Pollution
  - CO2 Emissions
  - Water and Electricity Consumption
  - Weather Conditions

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed

- Data inspection and cleaning
- Univariate and multivariate analysis
- Correlation analysis using heatmaps
- Outlier detection using the **IQR method**
- Temporal analysis of traffic congestion patterns

## Key Insights

- Traffic congestion peaks during **morning (8–10 AM)** and **evening (5–7 PM)**.
- Higher **renewable energy usage is associated with lower CO2 emissions**.
- **Traffic congestion and vehicle speed show an inverse relationship**.
- Significant variation in **PM2.5 and noise pollution levels** across cities.
